# erica_game_contest

## 색상 기억 게임 (Memory Color Game)

tkinter를 사용하여 구현한 색상 순서 기억 게임입니다. 중앙에 표시되는 색상 시퀀스를 기억하고, 주변 버튼을 올바른 순서로 클릭하여 점수를 획득하세요!

## 게임 설명

- **게임 방식**: 각 라운드마다 색상 시퀀스가 하나씩 추가됩니다
- **시퀀스 표시**: 중앙 라벨에 색상이 순서대로 표시됩니다
- **입력 방법**: 표시된 색상 순서대로 주변 색상 버튼을 클릭하세요
- **게임 진행**: 정답을 맞추면 다음 라운드로 진행되며, 오답을 입력하면 게임이 종료됩니다
- **점수**: 게임 종료 시 최종 점수와 마지막 색상 시퀀스가 출력됩니다

## 게임 화면 구성

- **3x3 그리드 레이아웃**
  - 중앙 (1, 1): 시퀀스 표시 영역 (흰색 라벨)
  - 주변 8개 위치: 색상 버튼 (빨강, 파랑, 초록, 노랑, 주황, 분홍, 갈색, 회색)


## 요구사항

- tkinter (일반적으로 Python에 기본 포함)

## 게임 규칙

1. 게임 시작 시 첫 번째 색상이 중앙 라벨에 표시됩니다
2. 표시가 끝나면 주변 색상 버튼이 활성화됩니다
3. 표시된 색상 순서대로 버튼을 클릭하세요
4. 모든 색상을 올바른 순서로 입력하면 다음 라운드로 진행됩니다
5. 다음 라운드에서는 이전 시퀀스에 새로운 색상이 추가되어 표시됩니다
6. 잘못된 색상을 클릭하면 게임이 종료되고 점수가 표시됩니다

## 점수 계산

- 점수 = `len(memory_color) - 1`
- 즉, 마지막 라운드의 시퀀스 길이에서 1을 뺀 값입니다

## 주요 기능

- 색상 시퀀스 자동 생성 및 표시
- 사용자 입력 검증
- 라운드별 난이도 증가
- 게임 종료 시 점수 및 시퀀스 출력
